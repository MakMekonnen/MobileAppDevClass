# MobileAppDevClass
For AVT-319-DL1 with Prof Harward

A responsive portfolio website showcasing photography, 2D animation compositing, and video editing work.
Built with semantic HTML, clean CSS, accessible design, and lightweight vanilla JavaScript.

---Features
Core Functionality

Responsive Layout optimized for mobile, tablet, and desktop

Semantic HTML5 Structure using header, nav, main, section, article, figure, and footer

Centered Contact Section with clean visual hierarchy

Flexbox Project Gallery with row-based image layouts

Interactive Enhancements (JavaScript)

Hamburger Mobile Menu with accessible toggling

Smooth Scrolling between sections

Fullscreen Lightbox when clicking any project image

Real-time Email Validation in the contact form
